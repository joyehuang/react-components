# Getting Started

## Prerequisites

- Node.js 20+
- pnpm 9+

## Install

```bash
pnpm install
```

## Run Apps

```bash
pnpm dev:playground
pnpm dev:docs
```

## Validate Manifest And Quality Gates

```bash
pnpm validate:manifest
pnpm quality:check
```
