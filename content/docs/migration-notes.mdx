# Migration Notes

## Architecture

- Existing Vite app is now hosted under `apps/playground`.
- Documentation moved to `apps/docs` with Fumadocs navigation.

## Data Source

- Legacy in-app registry is replaced by `packages/registry` manifest.
- Preview rendering stays app-specific while metadata remains centralized.

## Component API Direction

- Named exports are the preferred API surface.
- Pilot quality migration targets:
  - `CreditCard`
  - `TextScatter`

## Compatibility

- Legacy default exports can still exist temporarily.
- New integrations should import from `@rc-lab/ui` and `@rc-lab/registry`.
