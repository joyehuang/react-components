# RC LAB Documentation

This workspace ships a dual-app architecture:

- `apps/playground`: Vite app for rapid component development.
- `apps/docs`: Next.js + Fumadocs documentation site.

The catalog, docs, and CLI all consume the same manifest from `packages/registry`.
