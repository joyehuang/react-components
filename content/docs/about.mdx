# 关于项目

这是一个大学生通过 **Vibe Coding** 持续打磨的 React 组件库项目。

## 初心

临摹市面上优秀的开源/闭源组件效果，不停留在“演示”，而是沉淀成：

- 可复用
- 可复制
- 可通过 CLI 安装
- 可被文档化检索

## 项目特点

- 组件页面都有可视化预览
- 同时提供代码复制与命令安装
- 以 manifest 驱动文档、CLI 与质量门禁

## English

A React component library built by a college student via vibe coding, focused on productizing high-quality UI patterns for real reuse.
